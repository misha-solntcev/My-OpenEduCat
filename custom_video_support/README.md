# Custom Video Support (VK, Rutube)

Этот модуль расширяет возможности редактора Odoo 18 (HTML Editor), добавляя поддержку популярных видеохостингов **VK (ВКонтакте)** и **Rutube**.

## Основные возможности
* **Поддержка Rutube**: распознавание ссылок на видео (url) и кодов для вставки в плеер (iframe).
* **Поддержка VK & VKVideo**: работа с доменами `vk.com` и `vkvideo.ru`.
* **Автоматическое извлечение параметров**: корректная обработка `oid`, `id` и `hash` для видео из групп и закрытых альбомов VK.
* **Интеграция с Odoo 18**: использование новой архитектуры Embedded Components.
* **Исправление ошибок**: предотвращает ошибку "Unsupported platform", которая возникает в стандартной Odoo 18 при попытке вставить стороннее видео.

## Установка
1. Скопируйте папку модуля в вашу директорию `addons`.
2. Обновите список модулей в Odoo.
3. Установите модуль `custom_video_support`.

## Использование
1. Откройте любой редактор (Проекты, Задачи, Описание товара).
2. Нажмите /Ссылка на видео (Вставить видео).
3. Вставьте ссылку на видео (url) или код `<iframe>`. 
4. Нажмите **"Вставить видео"**. Видео появится в тексте и будет доступно для предпросмотра.

## Особенности и ограничения
* **Предпросмотр в диалоговом окне**: 
    * При вставке прямого **URL** (например, из адресной строки браузера) предпросмотр внутри окна выбора может не отображаться. Это связано с политикой безопасности видеохостингов (запрет на встраивание стандартных страниц для защиты от кликджекинга).
    * При вставке кода **<iframe>** (код для вставки) предпросмотр работает в полном объеме.
    * **Важно**: В обоих случаях после нажатия кнопки "Вставить видео" плеер будет корректно отображаться и работать внутри документа Odoo.
* **Браузер Firefox**: В этом браузере может потребоваться отключение "Улучшенной защиты от отслеживания" (иконка щита слева от URL) для того, чтобы плееры VK стали видимыми, так как Firefox по умолчанию блокирует социальные виджеты.
