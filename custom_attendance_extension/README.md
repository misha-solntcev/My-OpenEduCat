# Custom Attendance Extension

Этот пользовательский модуль расширяет функциональность модуля посещаемости OpenEduCat, добавляя поле "Тема урока".

## Функциональность

1. Добавляет поле "Тема урока" в форму ведомости посещаемости
2. Сохраняет темы уроков в соответствующих записях оценок по предметам
3. Позволяет просматривать темы уроков в форме оценок по предметам (при открытии карточки)

## Установка

1. Поместите этот модуль в каталог пользовательских модулей вашего проекта OpenEduCat
2. Обновите список модулей в интерфейсе Odoo
3. Установите модуль "Custom Attendance Extension"

## Использование

1. При заполнении ведомости посещаемости заполните поле "Тема урока"
2. При обновлении оценок по предметам темы уроков будут автоматически перенесены в соответствующие записи
3. Темы уроков можно просматривать в форме оценок по предметам (при открытии карточки)

## Технические детали

Модуль использует механизм наследования Odoo для расширения следующих моделей:
- `op.attendance.sheet` - добавлено поле `lesson_topic`
- `op.subject.grades` - добавлено поле `lesson_topics`

Также переопределен мастер обновления оценок по предметам для обработки тем уроков.
