# MAX Messenger Gateway

Gateway integration for MAX messenger (platform-api.max.ru) via Odoo mail_gateway.

## Features

- Register MAX bot as a mail gateway
- Receive messages from MAX users in Odoo Discuss
- Reply to MAX users from Odoo Discuss
- Webhook management (integrate / update / remove)
- Authorized chats whitelist
- Fetch Chats (discover chat IDs)
- Test Connection
