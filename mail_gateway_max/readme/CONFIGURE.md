1. Register at https://business.max.ru and verify your organisation (Russian legal entity required).
2. Create a bot in the business cabinet → get API token.
3. In Odoo: Settings → Technical → Email → Mail Gateway → Create.
4. Select type "MAX", paste the token.
5. Set a Webhook key and Webhook secret.
6. Save → press "Integrate Webhook".
7. Press "Fetch Chats" to discover chat IDs (requires webhook to be disabled temporarily).
