# Реализация цветных бейджей оценок

## Назначение
Цветные бейджи оценок в Kanban-представлении позволяют визуально быстро отличать оценки разного уровня. Каждая оценка отображается в виде цветного значка, что улучшает восприятие информации.

## Цветовая схема
В реализации используются следующие цвета для оценок:

- **Оценка 2**: Красный
  - Фон: #fff0f6
  - Текст: #d6336c

- **Оценка 3**: Желтый
  - Фон: #fff3bf
  - Текст: #997404

- **Оценка 4**: Зеленый
  - Фон: #ebfbee
  - Текст: #37b24d

- **Оценка 5**: Синий
  - Фон: #e7f5ff
  - Текст: #1c7ed6

- **Другие значения**: Серый
  - Фон: #f8f9fa
  - Текст: #495057

## Техническая реализация

### Шаблон QWeb
Цветные бейджи реализованы в шаблоне QWeb в файле `subject_grades_view.xml`. Используется конструкция `<t t-foreach>` для перебора оценок, разделенных запятыми.

### Обработка строк
Для корректной обработки строк в шаблоне QWeb используется подход:
```xml
<t t-set="final_mark" t-value="('' + mark).trim()"/>
```

Этот подход необходим, потому что в QWeb нельзя напрямую вызывать методы JavaScript на переменных из контекста. Выражение `('' + mark).trim()` преобразует значение в строку и удаляет пробелы по краям.

### Условное отображение
Для каждой оценки проверяется ее значение, и в зависимости от этого применяется соответствующий цвет. Используются условные конструкции `<t t-if>`, `<t t-elif>` и `<t t-else>`.

### Стилизация
Каждый бейдж стилизован с помощью inline CSS для обеспечения корректного отображения:
- `class="badge fw-bold"` - базовые стили бейджа с жирным шрифтом
- `style="background-color: #color; color: #color; font-size: 12px; margin-right: 2px;"` - индивидуальные стили для каждого цвета

### Расположение
Бейджи отображаются в нижней части карточки Kanban-представления, что позволяет сразу видеть все оценки студента по предмету.

## Особенности реализации
1. Используется правильный подход для обработки строк в QWeb
2. Реализована поддержка нечисловых значений оценок
3. Применяются стандартные классы Bootstrap для стилизации бейджей
4. Добавлены отступы между бейджами для лучшего визуального восприятия
5. Оценки отображаются жирным шрифтом для лучшей видимости
