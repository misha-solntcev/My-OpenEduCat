# Wizard обновления данных op.update.subject.grades

## Назначение
Wizard `op.update.subject.grades` предназначен для обновления агрегированных данных в модели `op.subject.grades` на основе записей посещаемости из модели `op.attendance.line`.

## Методы

### update_subject_grades
Основной метод wizard, выполняющий следующие действия:
1. Удаляет все существующие записи в модели `op.subject.grades`
2. Получает все записи посещаемости с непустыми оценками из поля `x_mark`
3. Группирует данные по студенту и предмету (используя поле `x_subject`)
4. Для каждой группы вычисляет:
   - Все оценки в виде строки
   - Среднюю оценку
   - Общее количество занятий
   - Количество посещенных занятий
   - Дату последнего занятия
5. Создает новые записи в модели `op.subject.grades`

## Использование
Wizard доступен в меню "Class Attendances" -> "Configuration" -> "Обновить оценки по предметам" только для пользователей с правами менеджера посещаемости.

## Особенности реализации
- Использует поле `x_subject` вместо `course_id` для определения предмета
- Обрабатывает только числовые значения оценок при вычислении средней оценки
- Преобразует все оценки в строку с разделением запятыми
- Учитывает только записи с непустым полем `x_mark`
